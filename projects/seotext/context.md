# seotext — AI SEO-копирайтер для РФ

Дата старта: 2026-04-21 (выбрано после глубокого ресёрча 15 идей)
Домен: TBD (купить на этой неделе — кандидаты: lsi.pro / smartseo.ru / textai.ru / nurilytics.ru / seokiller.ru / yapisat.ru)
Статус: **validation phase** — лендинг-тест без кода

## Что это

AI-копирайтер под Yandex-SEO с LSI-покрытием, кластеризацией семантики и встроенным антиплагиатом. Заменяет генералистов (Gerwin, TurboText) специализацией на русскоязычном SEO.

## ЦА

**Первичная:**
- Руководители/линейные контент-менеджеры SEO-агентств
- Размер агентства: 5-30 клиентов
- Объём контента: 20-100 статей/мес
- В РФ ~800-1094 таких агентств (Runet Awards, MARKETING-TECH каталог)

**Вторичная:**
- Копирайтеры-ИП/самозанятые с годовой выручкой 500k-1.5M ₽
- Специализация: Yandex-SEO (не ChatGPT-обёртки под Zen)
- В РФ десятки тысяч

## Value proposition

«AI-копирайтер для Yandex-SEO с LSI, кластеризацией и антиплагиатом. В 3× быстрее человека, в 10× дешевле Kwork-фрилансера.»

## Почему выбрано

1. Score 52/70 в матрице 8 ниш (highest)
2. Чёткая дыра: 3 из 10 проверенных конкурентов мертвы (textformer.ru, 24writer.ru, neuropod.ru); 7 живых — все универсальные GPT-обёртки, никто не закрывает LSI+Yandex+антиплагиат+кластеризацию в связке
3. Русскоязычный B2B — ЮKassa, ИП Даши, юрсхема отработаны в SecAudit
4. Каналы inbound концентрированы: Habr хаб SEO (300k+), searchengines.guru (10k DAU), vc.ru
5. ARPU 4-12k ₽/мес → 50-100 платящих = 150-200k MRR (достижимо)
6. Переиспользование кода SecAudit 60-70%

## MVP (5 фич для M1-M2)

1. **Генератор статей** 2000-8000 знаков: тема → LLM (Claude/DeepSeek/GigaChat) + sanity-check фактов
2. **LSI-анализ**: парсинг Yandex SERP по ключу (XMLRiver API), сравнение top-10 выдачи с генерацией, покрытие семантики
3. **Кластеризация семантики**: группировка запросов через embeddings + эвристики
4. **Антиплагиат встроенный**: shingle-based чекер + опц. интеграция text.ru API
5. **Экспорт**: DOCX + WebHook для WordPress/Tilda auto-publish

## Технический стек

**Переиспользование из SecAudit (60-70%):**
- `internal/ai/*` — Claude + DeepSeek провайдеры (+ добавить GigaChat/YandexGPT опцию для B2B)
- `internal/handler/auth.go` — регистрация, rate-limit, disposable email
- `internal/validation/*` — email, password, sanitize
- `internal/payment/yookassa/` — тарифы адаптировать
- `internal/repository/*` (users, profiles, subscriptions) — скелет
- Frontend Next.js 14 + Tailwind — 80%

**Новое:**
- Yandex SERP-парсер через XMLRiver/KeyCollector API (или собственный через прокси)
- LSI-матч-алгоритм (embeddings + semantic coverage)
- Кластеризация запросов (embedding similarity + агломеративная кластеризация)
- Antиплагиат (shingle + text.ru API)
- WP/Tilda WebHook публикации

## Юр-риски

- **Приём платежей:** ЮKassa/Tinkoff/Robokassa — штатно, ИП/ООО
- **152-ФЗ:** применим при регистрации, решено в SecAudit
- **Зарубежные LLM** (OpenAI/Claude): формально разрешены, но B2B-тендеры госсектора требуют GigaChat/YandexGPT — добавить как опцию в Agency
- **Авторское право** на сгенерированный текст: Минцифры пока не регулирует, ответственность на пользователе (дисклеймер)
- **Парсинг Yandex SERP** — серая зона (риск блокировки IP, не юр-претензия); использовать XMLRiver (готовый сервис данных)
