Дата: 2026-04-21 (вечер)
Контекст: проверка реалистичности заявленных 21-35 ч/неделю на продукты.
Связанные: `decisions/portfolio-strategy.md`, `decisions/anti-burnout.md`.

## Исходные вводные

- 40ч/нед — основная работа (Go dev, удалёнка)
- Локация: Россия, живу с Дашей (поддерживает стартап)
- Приоритет здоровья: пролактин 440 (норма 407), зал 4×/нед
- Заявлено: 3-5 ч/день на продукты = 21-35 ч/неделю

**Остаток на продукты: 27-38 ч/нед**

## Честный вердикт

Заявленные 21-35 ч/нед — верхняя граница ТЕОРЕТИЧЕСКИ вмещается, но НЕ sustainable:

- **22-26 ч/нед** — реально устойчиво месяцами без выгорания
- **27-30 ч/нед** — peak, держится 4-8 недель, требует компенсации
- **31-35 ч/нед** — только короткими спринтами (2-3 нед), после — обязательный отдых

Почему не 35ч/нед устойчиво:
1. **Энергия после 8ч брайнворка ≠ 100%.** Продуктовые часы — это ~70% продуктивности «утренней свежей головы». Формально 5ч вечером = реально 3.5ч качественной работы.
2. **Жизненный шум**: праздники, болезни, усталые недели, внеплановые дедлайны на работе. Реально теряется 3-5 недель/год.
3. **Пролактин требует стабильного режима.** Любой стресс/недосып → показатели растут. 35ч/нед ставит режим под риск.
4. **Зал 4×/нед возвращается** — это невостребуемые 6-7ч/нед, их нельзя «немного пропустить ради продукта».

## Рабочий бюджет для планирования

**Планируем стратегию на 24-26 ч/нед sustainable.**

Распределение по дням (пример):
- Пн-Чт: 2.5-3.5 ч/день после работы × 4 = 10-14 ч
- Пт: 0-2 ч (обычно отдых после рабочей недели)
- Сб: 5-7 ч (основная продуктовая сессия недели)
- Вс: **0 ч** (жёстко, без исключений — см. `anti-burnout.md`)

Итого: 15-23 ч будни + 5-7 ч сб = 20-30 ч, с таргетом 24-26 ч.

## Что это значит для стратегии

- **Нельзя планировать 2 продукта по 20ч каждый** (нужно 40ч/нед — нереалистично).
- **Можно:** 1 продукт основной (15-20ч) + 1 продукт low-touch (3-5ч) = 18-25ч.
- **ИЛИ** 1 продукт фокусно (22-26ч) и ноль второго.
- Время на маркетинг (контент, SEO) нельзя ужимать <50% продуктового бюджета — продукт без канала бесполезен.

Формула распределения (на 24ч/нед базовых):
- 50% маркетинг/контент/SEO (12ч): 2 статьи/мес + посты + SEO-оптимизация
- 30% продукт (7ч): баги, улучшения, фичи по запросу клиентов
- 20% customer/ops (5ч): поддержка, интервью, аналитика, финансы

## Когда пересматривать

- Пролактин снижается ниже 407 стабильно → можно +2-3 ч/нед
- Пролактин растёт/Даша выражает напряжение → −5 ч/нед немедленно
- Работа временно снижает нагрузку (отпуск, low-season) → +3-5 ч/нед краткосрочно
- Появляется обратная связь «выгораю» (см. red flags в `anti-burnout.md`) → −5-10 ч/нед
