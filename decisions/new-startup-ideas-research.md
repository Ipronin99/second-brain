- **Вывод:** повторение SecAudit vs Б-152.

### #2 Шаблонизатор для самозанятых — score 6.5/10 — УСЛОВНО

- **Рынок:** 15.27 млн самозанятых (ФНС), +25% YoY, доход 32.400 ₽/мес.

### #3 YouTube-креаторы — score 5/10 — ПРОПУСК

**Value prop:**
«AI-копирайтер для Yandex-SEO с LSI-покрытием, кластеризацией семантики и встроенным антиплагиатом. В 3× быстрее человека, в 10× дешевле Kwork-фрилансера.»

**MVP (5 фич):**
1. Генератор статей 2000-8000 знаков (Claude/GigaChat + факт-чекер)
2. LSI-анализ через парсинг Yandex SERP (сравнение top-10 выдачи с генерацией)
3. Кластеризация семантики (embeddings + клюк-группировка)
4. Антиплагиат (shingle-based + опц. text.ru API)
5. Экспорт DOCX + WebHook для WordPress/Tilda auto-publish

**Монетизация:**
- Free — 3 статьи/мес (lead magnet)
- Start 1990 ₽/мес — 30 статей + LSI + кластеризация
- Pro 4990 ₽/мес — 100 статей + антиплагиат + WP-WebHook + 2 seats
- Agency 12 900 ₽/мес — 500 статей + API + 5 seats
- Pay-per-article 99 ₽ — разовый
- Blended ARPU target: 3500-4500 ₽/мес

**План 6 мес (M1 = май 2026):**
- M1-M2: MVP 3 фич (генерация + LSI + антиплагиат). 15-20 free beta. Интервью.
- M3: + кластеризация + WP. Публичный запуск. 2 Habr-статьи + 1 searchengines. **5-10 платящих, 15-45k MRR.**

**Прогноз MRR:**
